# 正则表达式（Regular Expression）（`java.util.regex` 包）

`String` 类中有 `mathes(regex)` 方法，可以用正则表达式匹配

---

### 一、自定义字符类 `[]`

**字符类**用于匹配某个位置上的**一个字符**

| 表达式            | 匹配内容                                  |
| -------------- | ------------------------------------- |
| `[abc]`        | a、b 或 c（只匹配一个字符）                      |
| `[^abc]`       | 除 a、b、c 以外的任意一个字符（^在开头是否定）            |
| `[a-z]`        | a 到 z 之间的任意一个字符（范围）                   |
| `[a-zA-Z]`     | a到z A到Z 任意一个英文字母（大小写）                 |
| `[a-d[m-p]]`   | a到d或m到o                               |
| `[0-9]`        | 任意一个数字                                |
| `[a-z&&[def]]` | a到z和def。为：d，e，f                       |
| `[a-z&&[^bc]]` | a 到 z 中除去 b 和 c（交集与减法，`&&` 交集，`^` 排除） |

注意：在 `[]` 内部大多数字符不需要转义，但 `\`、`^`（如果不在开头）、`-`（如果在中间要表示字面量）等仍需转义或用位置规避。

---

### 二、预定义字符类

预定义字符类是常用字符类的简写

| 表达式    | 匹配内容                       | 等价写法                  |
| ------ | -------------------------- | --------------------- |
| `.`    | 任意一个字符（可能不匹配换行符，取决于标志）     | —                     |
| `\d`   | 一个数字                       | `[0-9]`               |
| `\D`   | 一个非数字                      | `[^0-9]`              |
| `\w`   | 一个单词字符（字母、数字、下划线）          | `[a-zA-Z0-9_]`        |
| `\W`   | 一个非单词字符                    | `[^\w]`               |
| `\s`   | 一个空白字符（空格、制表符、换行等）         | `[\t\n\x0B\f\r]`      |
| `\S`   | 一个非空白字符                    | `[^\s]`               |
| `\b`   | 单词边界                       |                       |
| `\B`   | 非单词边界                      |                       |
| `^`    | 行首（在 `[]` 外），或取反（在 `[]` 内） | `^abc`，`[^0-9]` 表示非数字 |
| `$`    | 行尾                         | `abc$`                |
| `\`    | 转义                         |                       |
| `\|`   | 或者                         | `cat\|dog`            |
| `()`   | 分组，同时捕获                    | `(ab)+`               |
| `(?:)` | 非捕获分组                      | `(?:ab)+` 只分组不保存      |

> Java 字符串中**必须双重转义**：写成 `"\\d"`、`"\\w"` 等，否则编译器会将 `\d` 识别为无效转义符。

### 三、数量词

数量词用于指定**前一个字符或分组**出现的次数。在 Java 正则中，数量词分为三种模式：**贪婪（Greedy）**、**勉强/非贪婪（Reluctant/Lazy）**、**占有（Possessive）**。

---

#### 1、基本数量词

| 数量词     | 匹配次数       | 说明                   |
| ------- | ---------- | -------------------- |
| `?`     | 0 或 1 次    | X?，X出现一次或者零次         |
| `*`     | 0 或多次      | X*，X出现任意次，包括零次       |
| `+`     | 1 或多次      | X+，X出现至少一次           |
| `{n}`   | 恰好 n 次     | X{n}，X出现恰好n次         |
| `{n,}`  | 至少 n 次     | X{n,}，X出现至少n次        |
| `{n,m}` | n 到 m 次（含） | X{n,m}，X出现至少n次，不超过m次 |

---

#### 2、贪婪、勉强、占有模式

| 模式             | 写法                      | 匹配逻辑                                  |
| -------------- | ----------------------- | ------------------------------------- |
| 贪婪（Greedy）     | `*`、`+`、`?`、`{n,m}`     | 尽可能多地匹配，但允许回溯以满足整体匹配                  |
| 勉强（Reluctant）  | `*?`、`+?`、`??`、`{n,m}?` | 尽可能少地匹配，先尝试忽略再逐次增加                    |
| 占有（Possessive） | `*+`、`++`、`?+`、`{n,m}+` | 尽可能多地匹配，**不回溯**；一旦匹配了就不会释放，可能导致整体匹配失败 |

**例**：对字符串 `"aaaa"` 应用正则 `a*`（贪婪）匹配整个字符串；`a*?`（勉强）匹配空字符串；`a*+`（占有）也匹配整个字符串，但若正则后面还有 `a` 导致需要回溯则会失败。
