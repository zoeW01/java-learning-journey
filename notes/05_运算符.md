# 算术运算符：

- 加 减 乘 除 取余

```java
int a = 10;
int b = 3;
System.out.println(a + b);   // 13  加
System.out.println(a - b);   // 7   减
System.out.println(a * b);   // 30  乘
System.out.println(a / b);   // 3   除（整数相除得整数，小数被丢弃）
System.out.println(a % b);   // 1   取余（10除以3余1）
```

# 比较运算符（结果一定是 `boolean`）：

- 大于小于，大于等于，小于等于，等等不等

```java
System.out.println(a > b);   // true
System.out.println(a < b);   // false
System.out.println(a >= b);   // true
System.out.println(a <= b);   // false
System.out.println(a == b);  // false  注意：判断相等用两个等号
System.out.println(a != b);  // true   不等于
```

**特别注意：整数除法**

```java
System.out.println(10 / 3);   // 输出 3，不是 3.3333
System.out.println(10.0 / 3); // 输出 3.3333333333333335
```

- 只要除号两边有一个是小数，结果就是小数。

# 自增自减运算法符


**核心规则**：
- `i++`：先取i当前值参与表达式运算，再自增。
- `++i`：先自增，再取新值参与表达式运算。
