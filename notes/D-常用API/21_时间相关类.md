# Date (`java.util.Date`)

Date 表示从 **1970年1月1日 00:00:00 GMT** 开始经过的时间。

|  返回值   |          方法          |                        说明                        |
| :----: | :------------------: | :----------------------------------------------: |
|  构造方法  |       `Date()`       |             创建表示**当前系统时间**的 Date 对象              |
|  构造方法  | `Date(long millis)`  | 根据**指定毫秒值**创建 Date 对象（从1970-01-01 00:00:00 GMT起） |
| `long` |     `getTime()`      |           **获取 Date 对象对应的毫秒值**（自纪元时间起）           |
| `void` | `setTime(long time)` |            **设置 Date 对象的毫秒值**（覆盖原时间）             |

# SimpleDateFormat（`java.text.SimpleDateFormat`）

用于**日期格式化与解析**的工具类

|   返回值    |                 方法                 |             说明             |
| :------: | :--------------------------------: | :------------------------: |
|   构造方法   |        `SimpleDateFormat()`        | 默认模式和默认区域（1970/1/1 上午8:00） |
|   构造方法   | `SimpleDateFormat(String pattern)` |     **自定义格式**和**默认区域**     |
| `String` |        `format(Date date)`         | 将 `Date` 对象转换为**指定格式的字符串** |
|  `Date`  |       `parse(String source)`       |   将**字符串解析为 `Date` 对象**    |
