
# 包装类

![Java Number类](https://www.runoob.com/wp-content/uploads/2013/12/OOP_WrapperClass.png)

1. **包装类的定义：**包装类就是为了解决基本类型不能直接参与面向对象开发的问题，使基本类型通过装箱成引用数据类型的形式存在。
    
    - 八个包装类对应八种基本类型
        
    - `Byte`、`Short`、`Integer`、`Long`、`Float`、`Double`、`Boolean`、`Character`
        
    - `Character`和`Boolean`是直接继承自`Object`类的，其余六个包装类(都是数字类)继承自`Number`类
        
    - 包装类都重写了`equals`.
        
2. **包装类的语法：**以前的`JDK`可以用`new`的方式创建包装类，现在需要借助包装类的静态方法`valueOf`创建。
    
    - `static valueOf`方法会复用一个字节范围内(-128~127)的数据(缓存在数字常量池内)
        
    - 八种包装类中，只有两个小数类型没有缓存，`Character`缓存0~127，`Boolean`缓存`true`和`false`
        
3. **自动拆装箱：**也就是基本类型和包装类型的互转。`JDK1.5`提供了自动拆装箱。当编译器编译时若发现是基本类型与包装类型之间的相互赋值，则自动补齐代码完成转换工作，这个过程叫做自动拆装箱。
    
    - 自动装箱本质调用`valueOf()`
        
    - 自动拆箱本质调用`xxxValue()`
        
    
    Integer i=Integer.valueOf(number);-----------------Integer i=number;  
    //自动装箱(隐藏了valueOf方法，自动补齐代码)
    
    int number=i.intValue------------------------------int number=i;  
    //自动拆箱(隐藏了xxxValue方法，自动补齐代码)
    
    - 自动装箱使用`valueOf()`会利用缓存机制，直接`new`时会绕过缓存总是创建新对象。
        
    - **包装类常用操作：** 1)可以通过包装类来获取基本类型的取值范围:
        
        int max = Integer.MAX_VALUE; //获取int的最大值  
        int min = Integer.MIN_VALUE; //获取int的最小值  
          
        Long max1 = Long.MAX_VALUE; //获取long的最大值  
        long min1 = Long.MIN_VALUE; //获取long的最小值
        
        2)通过包装类可以将字符串转换为对应的基本类型：得是一个正常的对应类型(在对应取值范围内)(`static parse`方法)（Character类型没有对应的转换方式）
        
        String str = "对应字符串;  
        byte b = Byte.parseInt(str);  
        System.out.println(b);  
          
        String str = "对应字符串;  
        short s = Short.parseInt(str);  
        System.out.println(s);  
          
        String str = "对应字符串;  
        int i = Integer.parseInt(str);  
        System.out.println(i);  
          
        String str = "对应字符串;  
        long l = Long.parseLong(str);  
        System.out.println(l);  
          
        String str = "对应字符串;  
        float f = Float.parseFloat(str);  
        System.out.println(f);  
          
        String str = "对应字符串;  
        double d = Double.parseDouble(str);  
        System.out.println(d);  
          
        String str = "对应字符串;  
        boolean bo = Boolean.parseBoolean(str);  
        System.out.println(bo);  
          
        String str = "对应字符串;  
        Character/char c = str.charAt(0);  
        System.out.println(c);  
        //Character/char 没有对应的parse方法
