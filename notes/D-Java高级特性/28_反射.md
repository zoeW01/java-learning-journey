# 反射

![img|45](https://img2020.cnblogs.com/blog/1971784/202003/1971784-20200318213539251-923984642.png)

1. 什么是反射？JAVA反射机制是在运行状态中，对于任意一个类，都能够知道这个类的所有属性和方法；对于任意一个对象，都能够调用它的任意一个方法，这种动态获取、调用对象方法的功能称为java语言的反射机制。反射是通过Class对象(字节码文件)，来知道某个类的所有属性和方法。也就是说通过反射我们可以获取构造器，对象，属性，方法（原本不知道）
    
2. 什么是反射机制：反射是动态机制，反射是Java语言在运行时（Runtime）实现的一种元编程（Meta - Programming）能力。
    
3. Java的反射功能主要通过以下几个类和接口实现：
    
    - `Class`：用于获取类或接口的信息。
        
    - `Field`：用于获取和设置类的字段。
        
    - `Method`：用于获取类的方法。
        
    - `Constructor`：用于获取类的构造函数。
        
    - `Array`：用于操作数组（通常不被视为反射对象）。
        
    - `Modifier`：用于获取字段、方法和类的修饰符
        

## Class类

1. **类对象**：Java的Class类是发射机制的基石，Class的每一个实例用于表示JVM中加载的一个类的信息。当JVM加载一个类时，JVM会创建对应的`Class`对象，该对象包含类的完整元数据。每个类在JVM中仅存在一个`Class`对象。
    
2. 反射操作的第一步就是获取要操作的类的类对象。JVM在使用一个类时，会读取该类的字节码文件，然后实例化一个Class对象记录加载的这个类的信息，因此在JVM内部每个被加载的类都有且仅有一个Class的实例与之关联，这个Class的实例称为类对象。通过类对象可以反映出其表示的类的一切信息(类名，包信息，构造器信息，方法信息，属性信息)，从而在程序运行期间进行动态调用。
    
3. 获取Class对象
    
    - **`类名.class` 语法（最安全、性能最好）**
        
        - 任何数据类型，以及每个类创建后都会有一个默认的静态的class属性用于返回该类的class对象
            
        - 基本数据类型获取类对象仅有这一种方式，基本类型对应的`Class`对象为`包装类.TYPE`。
            
        - 通过该类的对象获取对应的Class对象
            
    - **`对象.getClass()` 方法**
        
        - 该方法基本不用，显然和反射机制相悖（你有类对象还用反射获取Class类对象干嘛，多此一举）
            
    - **`Class.forName("全限定类名")`（最常用）**
        
        - 通过字符串形式的完整类名动态加载类，常用于配置文件中。**该方法会触发类的初始化**（执行静态代码块）。
            
        - **`public static Class<?> forName(String className)`**：**作用**：通过类的全限定名动态加载类并返回其`Class`对象。
            
    - **通过类加载器 `ClassLoader`（了解即可）**
        
4. Class类的常用方法：
    
    > 方法名加没有加`s`的返回的都是单个，加`s`的返回的都是所有；方法名包含`Declared`返回自身所有的，不包含`Declared`只返回自身和继承的公开
    
    1. 访问字段：
        
        - **`public Field getField(String fieldName)`**：**作用**：获取类中对应属性名的**公共字段**（包括继承的字段）。
            
        - **`public Field getDeclaredField(String fieldName)`**：**作用**：获取类**自身声明的对应属性名的字段**（包括私有字段）。
            
        - **`public Field[] getFields()`**：**作用**：获取类所有**公共字段**（包括继承的字段）。
            
        - **`public Field[] getDeclaredFields()`**：**作用**：获取类**自身声明的所有字段**（包括私有字段）。
            
    2. 获取方法：
        
        - **`public Method getMethod(String methodName，Class argumentName)`**：**作用**：获取类中对象方法名的**公共方法**（包括继承的方法）。
            
        - **`public Method getDeclaredMethod(String methodName，Class argumentName)`**：**作用**：获取类**自身声明的对应方法名方法**（包括私有方法）。
            
        - **`public Method[] getMethods()`**：**作用**：获取类所有**公共方法**（包括继承的方法）。
            
        - **`public Method[] getDeclaredMethods()`**：**作用**：获取类**自身声明的所有方法**（包括私有方法）。
            
    3. 调用构造方法：
        
        - **`@Deprecated public Object newInstance()`**：**作用**：通过无参构造方法创建类的实例（需构造方法为`public`且无参）。
            
        - **`public Constructor<T> getConstructor(Class<?>... parameterTypes)`**：**作用**：获取类的**公共构造方法**（参数类型为Class类，根据传参匹配对应构造器。无参构造器参数就是null或者空）。
            
        - **`public Constructor getConstructor(Class...)`**：**作用**：获取某个`public`的`Constructor`；
            
        - **`public Constructor getDeclaredConstructor(Class...)`**：**作用**：获取某个`Constructor`（包括私有构造器）；
            
        - **`public Constructor<?>[] getConstructors()`**：**作用**：获取类所有**公共构造方法**。
            
        - **`public Constructor<?>[] getDeclaredConstructors()`**：**作用**：获取类**所有构造方法**（包括私有构造方法）。
            
    4. 继承关系：
        
        - **`public Class<?> getSuperclass()`**：**作用**：获取直接父类的`Class`对象。（对所有`interface`的`Class`调用`getSuperclass()`返回的是`null`，获取接口的父接口要用`getInterfaces()`）
            
        - **`public Class<?>[] getInterfaces()`**：**作用**：获取类直接实现的所有接口。
            
        - **`public boolean isInstance(Object obj)`**：**作用**：判断指定对象是否是该 `Class` 对象所表示类或接口的实例（或者是其子类/实现类的实例）。
            
        - **`public boolean isAssignableFrom(Class<?> cls)`**：**作用**：判断当前类是否可以由参数类赋值（即当前类是参数类的父类或接口）。
            
    5. 获取属性：
        
        1. **`public String getName()`**：返回类的**完全限定名**（包含包路径）。
            
        2. **`public String getSimpleName()`**：**作用**：返回类的**简单名称**（不包含包路径）
            
        3. **`public Package getPackage()`**：**作用**：获取类所属的包信息。
            
        4. **`public int getModifiers()`**：**作用**：返回类的修饰符的整数表示。
            
        5. **`public String toString()`**：**作用**：返回类的字符串表示，格式为：
            
            [修饰符] class/interface 名称
            
    6. 判断：
        
        - **`public String toGenericString()`**：**作用**：返回包含泛型信息和修饰符的详细字符串表示。
            
        - **`public boolean isInterface()`**： **作用**：判断是否为接口。
            
        - **`public boolean isArray()`**：**作用**：判断是否为数组类型。
            
        - **`public boolean isEnum()`**：**作用**：判断是否为枚举类型。
            
        - **`public boolean isPrimitive()`**：**作用**：判断是否为基本数据类型
            
        - **`public ComponentType[] getComponentType()`**：**作用**：若为数组类型，返回数组元素的类型。
            
    7. 注解：
        
        - **`public boolean isAnnotation()`**：**作用**：判断是否为注解类型。
            
        - **`public boolean isAnnotationPresent(Class<? extends Annotation> annotationClass)`**：**作用**：判断类是否包含指定注解。
            
        - **`public <T extends Annotation> T getAnnotation(Class<T> annotationClass)`**：**作用**：获取该元素上指定类型注解，不存在返回null。
            
        - **`public Annotation[] getAnnotations()`**：**作用**：获取该元素上所有注解（包含继承的）。
            
        - **`public <T extends Annotation> T[] getAnnotationsByType(Class<T> annotationClass)`**：**作用**：返回与此元素关联的指定类型的所有注解（包括继承的）
            
        - **`public <T extends Annotation> T getDeclaredAnnotation(Class<T> annotationClass)`**：**作用**：返回直接存在于此元素上的指定类型的注解（忽略继承的注解）
            
        - **`public <T extends Annotation> T[] getDeclaredAnnotationsByType(Class<T> annotationClass)`**：作用：返回直接存在于此元素上的指定类型的所有注解（忽略继承的注解）
            
        - **`public Annotation[] getDeclaredAnnotations()`**：**作用**：获取类上**自身声明的所有注解**（不包含继承的）。
            
    8. 获取类加载器：**`public ClassLoader getClassLoader()`**：**作用**：获取类加载器。
        

## Filed类

Field是反射对象之一，它的每一个实例用于表示某个类中的某个属性

- `getName()`：返回字段名称；
    
- `getType()`：返回字段类型，也是一个`Class`实例；
    
- `getModifiers()`：返回字段的修饰符，它是一个`int`类型，不同的值表示不同的含义。
    
    - Modifier.isFinal(m);
        
    - Modifier.isPublic(m);
        
    - Modifier.isProtected(m);
        
    - Modifier.isPrivate(m);
        
    - Modifier.isStatic(m);
        
- `Field.set(Object obj, Object)`：修改给定对象对应字段的属性。第一个`Object`参数是指定的实例，第二个`Object`参数是待修改的值。
    
- `Field.get(Object)`：获取给定对象对应字段的属性。`Object`参数是指定的实例。
    
- 注解：
    
    - **`public boolean isAnnotation()`**：**作用**：判断是否为注解类型。
        
    - **`public boolean isAnnotationPresent(Class<? extends Annotation> annotationClass)`**：**作用**：判断类是否包含指定注解。
        
    - **`public <T extends Annotation> T getAnnotation(Class<T> annotationClass)`**：**作用**：获取该元素上指定类型注解，不存在返回null。
        
    - **`public Annotation[] getAnnotations()`**：**作用**：获取该元素上所有注解（包含继承的）。
        
    - **`public <T extends Annotation> T[] getAnnotationsByType(Class<T> annotationClass)`**：**作用**：返回与此元素关联的指定类型的所有注解（包括继承的）
        
    - **`public <T extends Annotation> T getDeclaredAnnotation(Class<T> annotationClass)`**：**作用**：返回直接存在于此元素上的指定类型的注解（忽略继承的注解）
        
    - **`public <T extends Annotation> T[] getDeclaredAnnotationsByType(Class<T> annotationClass)`**：作用：返回直接存在于此元素上的指定类型的所有注解（忽略继承的注解）
        
    - **`public Annotation[] getDeclaredAnnotations()`**：**作用**：获取类上**自身声明的所有注解**（不包含继承的）。
        
- AccessibleObject类的方法：
    
    - **`void setAccessible(boolean flag)`**
        
        - 设置此反射对象的可访问标志，允许绕过 Java 语言访问控制检查
            
    - **`boolean isAccessible()`**
        
        - 获取此反射对象的可访问标志值
            
    - **`static void setAccessible(AccessibleObject[] array, boolean flag)`**
        
        - 批量设置一组反射对象的可访问标志
            

## Method类

Method也是反射对象，它的每一个实例用于表示类中的某个方法

- `getName()`：返回方法名称；
    
- `getReturnType()`：返回方法返回值类型，也是一个Class实例；
    
- `getParameterTypes()`：返回方法的参数类型，是一个Class数组；
    
- getParameterCount(); //获取参数个数
    
- `getModifiers()`：返回方法的修饰符，它是一个`int`，不同的值表示不同的含义。
    
- `getExceptionTypes()` - 返回方法声明的异常类型数组
    
- `invoke(Object,argument)`就相当于调用该方法，`invoke`的第一个参数是对象实例，即在哪个实例上调用该方法，后面的可变参数要与方法参数一致，否则将报错。调用静态方法时，第一个参数永远为`null`
    
- 注解：
    
    - **`public boolean isAnnotation()`**：**作用**：判断是否为注解类型。
        
    - **`public boolean isAnnotationPresent(Class<? extends Annotation> annotationClass)`**：**作用**：判断类是否包含指定注解。
        
    - **`public <T extends Annotation> T getAnnotation(Class<T> annotationClass)`**：**作用**：获取该元素上指定类型注解，不存在返回null。
        
    - **`public Annotation[] getAnnotations()`**：**作用**：获取该元素上所有注解（包含继承的）。
        
    - **`public <T extends Annotation> T[] getAnnotationsByType(Class<T> annotationClass)`**：**作用**：返回与此元素关联的指定类型的所有注解（包括继承的）

    - **`public Annotation[] getDeclaredAnnotations()`**：**作用**：获取类上**自身声明的所有注解**（不包含继承的）。
        
- AccessibleObject类的方法：
    
    - **`void setAccessible(boolean flag)`**
        
        - 设置此反射对象的可访问标志，允许绕过 Java 语言访问控制检查
