### 一、数组的基本概念（特点）
- 数组是引用类型，用`new`在堆内存中分配空间。  

- 数组创建后长度**不可变**，可以通过`length`属性获取长度。  

- 数组是一个容器，里面可以放多个**同类型**的数据。

- 每个数据叫“元素”，通过**索引（下标）访问，索引从 0 开始，** 越界抛出`ArrayIndexOutOfBoundsException`。   

- 所有元素自动初始化为默认值

### 二、数组的声明与初始化

**方式一：静态初始化（知道存什么值）**

- `数据类型[] 数组名 = {值1, 值2, ...};`
- `数据类型[] 数组名 = new 数据类型[]{值1, 值2, ...};`

```java
int[] scores = {95, 88, 76, 64, 59};
String[] names = {"Zoe", "Tom", "Jerry"};
int[] arr = new int[]{1, 2, 3, 4, 5}; 
```

**方式二：动态初始化（不知道值，先占位置）**

- `数据类型[] 数组名 = new 数据类型[长度];`

```java
int[] scores = new int[5];  // 可以放5个int，默认全是0
scores[0] = 95;
scores[1] = 88;
// ...
```


- 致命点
	`int a[]` 兼容C风格，同样合法
	
	`int[] a = new int[3]{1,2,3};` **编译错**。指定长度就不能同时写静态初始化列表。
	
	`int a[3];` Java中不允许在声明时指定大小，方括号里不能写数字。这个只在C语言合法。

### 三、访问和修改元素

- 数组名记录的是数字首地址，下标从**0**开始，因为下标本质是内存偏移量，0表示首地址偏移0字节
	- 数组是引用类型，数组名存的是堆上数组对象的**地址**

### 四、数组长度

```java
int[] scores = {95, 88, 76};
System.out.println(scores.length);   // 3
```

`.length` 属性，返回数组能存多少个元素，不是最后一个元素的索引。最后一个元素的索引是 `length - 1`。

注意：字符串是`length()`方法，数组是`length`属性，。

### 五、遍历数组


**普通for遍历**：
```java
int[] scores = {95, 88, 76, 64, 59};

for (int i = 0; i < scores.length; i++) {
    System.out.println("第" + (i + 1) + "个成绩: " + scores[i]);
}
```
数组和 for 循环是绝配——用 `.length` 控制循环次数，用 `i` 当索引逐个访问元素。

**增强for遍历**：

```java
int[] scores = {95, 88, 76, 64, 59};

for (int x : scores) {
    System.out.println(x);
}
```
增强型for循环与普通for循环有什么区别？

- 普通for可以通过下标访问和修改元素，可以控制遍历范围。  
- 增强for（for-each）语法简洁，但不能获取下标，不能修改数组元素值（获取的是值的副本）。  
- 增强for适用于全部遍历、只读的场景。

### 六、二维数组

#### 二维数组的声明与创建

**声明**：
```java
int[][] arr;
int arr[][];   // 也合法
int[] arr[];   // 也合法，但可读性差
```

**静态初始化**：
```java
int[][] arr = {{1,2}, {3,4,5}, {6}};
```
三个一维数组，长度分别为2、3、1。

**动态初始化**：
```java
int[][] arr = new int[3][4];  // 3行4列，所有元素默认0
int[][] arr = new int[3][];   // 合法，3个空引用，每行稍后可单独创建
arr[0] = new int[2];
arr[1] = new int[3];
arr[2] = new int[1];
```
**选择必考**：`new int[3][4]` 是合法，`new int[][4]` 不合法（高维不能缺，低维可缺）。
