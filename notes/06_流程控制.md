# 一、控制的分类

所有程序逻辑只有三种结构：
- **顺序**：代码从上到下一行一行执行
- **分支**：根据条件选择执行不同代码
- **循环**：重复执行某段代码

# 二、if条件判断

编程的核心是让代码根据不同条件执行不同逻辑。`if` 语句就是做这个的。

---

## 2.1 if 语句的基本结构

```java
if (条件) {
    // 条件为 true 时执行
}
```

**条件必须是 boolean 类型**——要么 `true`，要么 `false`。

## 2.2 if-else

```java
if (条件) {
    // 条件为true时执行
} else {
    // 前面条件为false时执行
}
```

## 2.3、if-else if-else（多条件分支）

```java
if (条件) {
    // 条件为true时执行
} else if (另一个条件) {
    // 第一个false且这个true时执行
} else {
    // 前面所有条件都为false时执行
}
```

# 三、for 循环

- 程序中经常需要重复执行某段代码，比如打印1到100的数字。手动写100行 `System.out.println` 是愚蠢的。循环就是用来解决这个问题的。

## 3.1 for 循环的结构

```java
for (初始化; 条件; 更新) {
    // 循环体：条件为 true 时重复执行
}
```

- **初始化**：循环开始前执行一次，通常用来声明计数器变量
    
- **条件**：每次循环开始前检查，为 `true` 就执行循环体，为 `false` 就退出
    
- **更新**：每次循环体执行完后执行，通常用来让计数器增加

- **执行顺序分析**：
	1. 初始化（只执行一次）
	2. 判断条件（true进循环体，false结束循环）
	3. 执行循环体
	4. 执行迭代
	5. 回到第2步

# 四、while 循环与 do-while 循环

## 4.1 while循环

```java
while (条件) {
    // 先判断，条件true才执行。可能一次都不执行。
}
```

- **执行逻辑：** 先检查条件，条件为 true 就执行循环体，执行完再检查条件，直到条件为 false。

## 4.2 do-while循环

```java
do {
    // 先执行一次，再判断。至少执行一次。
} while (条件);
```

- 和 while 的唯一区别：**do-while 至少执行一次**，因为条件检查在循环体之后。

# 六、多分支：switch

**语法**
```java
switch (值) {
    case 值1:
        // 代码
        break;
    case 值2:
        // 代码
        break;
    default:
        // 以上case全不命中时执行
}
```

**支持的数据类型**（到Java 7为止）：
`byte`、`short`、`int`、`char`、`String`（Java 7+）、`enum`。

**关键**：`float`、`double`、`long`、`boolean` **都不行**。

**`break`的穿透效应**：

```java
int x = 2;
switch (x) {
    case 1: System.out.print("A");
    case 2: System.out.print("B");
    case 3: System.out.print("C");
    default: System.out.print("D");
}
// 输出：BCD
// 因为没有break，从case 2一直执行到default
```

**穿透陷阱**：如果case后面没写break，代码会**继续执行下一个case的代码**，直到碰到break或switch结束。每个case结束必须写break，除非你故意用穿透。

**`default`位置**：可以放在任意位置，不是必须在最后。但如果放在前面且没写`break`，照样穿透。

# 七、跳转语句

**四个跳转语句**：

| 语句               | 作用                                               |
| ---------------- | ------------------------------------------------ |
| `break`          | 完全终止循环或`switch`，程序跳出当前层继续执行后面的语句。带标签时可作用于外层循环。   |
| `continue`       | 结束本次循环迭代，跳过循环体中剩余语句，直接进入下一次循环的条件判断。带标签时可作用于外层循环。 |
| `return`         | 结束方法，返回调用处                                       |
| `System.exit(0)` | 终止JVM                                            |
**带标签的跳转**（多重循环中区分跳出哪一层）：

```java
outer:
for (int i=1; i<=3; i++) {
    for (int j=1; j<=3; j++) {
        if (i*j == 4) break outer;
    }
}
// 跳出外层循环，不是只跳内层
```

不带标签的`break`只能跳出当前最内层循环。带标签可以指定跳出外层，标签必须写在循环前面。选择题给你嵌套循环问跳出后`i`和`j`的值，是常见出题形式。

**`continue`标签同理**：`continue outer;` 结束外层循环的本次迭代，回到外层的迭代语句。
