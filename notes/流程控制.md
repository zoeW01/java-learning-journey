# 一、控制的分类

所有程序逻辑只有三种结构：
- **顺序**：代码从上到下一行一行执行
- **分支**：根据条件选择执行不同代码
- **循环**：重复执行某段代码

# 二、if条件判断

编程的核心是让代码根据不同条件执行不同逻辑。`if` 语句就是做这个的。

---

# 2.1 if 语句的基本结构

```java
if (条件) {
    // 条件为 true 时执行
}
```

**条件必须是 boolean 类型**——要么 `true`，要么 `false`。

## 2.2 if-else

```java
if (条件) {
    // 条件为true时执行
} else {
    // 前面条件为false时执行
}
```

## 2.3、if-else if-else（多条件分支）

```java
if (条件) {
    // 条件为true时执行
} else if (另一个条件) {
    // 第一个false且这个true时执行
} else {
    // 前面所有条件都为false时执行
}
```

# 三、for 循环

- 程序中经常需要重复执行某段代码，比如打印1到100的数字。手动写100行 `System.out.println` 是愚蠢的。循环就是用来解决这个问题的。

## 3.1 for 循环的结构

```java
for (初始化; 条件; 更新) {
    // 循环体：条件为 true 时重复执行
}
```

- **初始化**：循环开始前执行一次，通常用来声明计数器变量
    
- **条件**：每次循环开始前检查，为 `true` 就执行循环体，为 `false` 就退出
    
- **更新**：每次循环体执行完后执行，通常用来让计数器增加

- **执行顺序分析**：
	1. 初始化（只执行一次）
	2. 判断条件（true进循环体，false结束循环）
	3. 执行循环体
	4. 执行迭代
	5. 回到第2步

# 四、while 循环与 do-while 循环

## 4.1 while循环

```java
while (条件) {
    // 先判断，条件true才执行。可能一次都不执行。
}
```

- **执行逻辑：** 先检查条件，条件为 true 就执行循环体，执行完再检查条件，直到条件为 false。

## 4.2 do-while循环

```java
do {
    // 先执行一次，再判断。至少执行一次。
} while (条件);
```

- 和 while 的唯一区别：**do-while 至少执行一次**，因为条件检查在循环体之后。
