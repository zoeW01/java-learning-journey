
- 多态是继承的延伸，它让同一个方法在不同子类中表现出不同的行为。这是面试必问的核心概念，不夸张地说，不理解多态就等于没学面向对象。

- **核心机制：**

	- 父类引用可以指向子类对象
	    
	- 调用方法时，JVM 根据对象的**实际类型**决定执行哪个版本（动态绑定）。
	    
	- **编译看左边，运行看右边**——编译器检查父类有没有这个方法，JVM 运行时找子类重写的方法。
	    
	- 调用属性时，JVM 根据对象的引用类型决定执行哪个版本（静态绑定）。
		- **子类声明一个和父类同名的字段时，父类的字段并没有消失。对象内存里，两个字段同时存在**。


**重写（Override）与重载（Overload）的区别：**

- 重载：同一个类里，方法名相同，参数列表不同
    
- 重写：子类重新定义父类的同名方法，方法签名完全一致，但行为不同
    

---


**多态三要素：**

1. **继承**
    
2. **重写**
    
3. **父类引用指向子类对象**
