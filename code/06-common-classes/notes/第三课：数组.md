### 一、数组的基本概念

- 数组是一个容器，里面可以放多个**同类型**的数据。

- 每个数据叫“元素”，通过**索引（下标）**访问，索引从 0 开始。

- 数组创建后长度**不可变**

### 二、数组的声明与初始化

**方式一：静态初始化（知道存什么值）**

- `数据类型[] 数组名 = {值1, 值2, ...};`

```java
int[] scores = {95, 88, 76, 64, 59};
String[] names = {"Zoe", "Tom", "Jerry"};
```

**方式二：动态初始化（不知道值，先占位置）**

- `数据类型[] 数组名 = new 数据类型[长度];`

```java
int[] scores = new int[5];  // 可以放5个int，默认全是0
scores[0] = 95;
scores[1] = 88;
// ...
```

**方式三**

- `数据类型[] 数组名 = new 数据类型[]{值1, 值2, ...};`

### 三、访问和修改元素

- 数组名记录的是数字首地址，下标从**0**开始，因为下标本质是内存偏移量，0表示首地址偏移0字节
	- 数组是引用类型，数组名存的是堆上数组对象的**地址**

### 四、数组长度

```java
int[] scores = {95, 88, 76};
System.out.println(scores.length);   // 3
```

`.length` 返回数组能存多少个元素，不是最后一个元素的索引。最后一个元素的索引是 `length - 1`。

### 五、遍历数组

```java
int[] scores = {95, 88, 76, 64, 59};

for (int i = 0; i < scores.length; i++) {
    System.out.println("第" + (i + 1) + "个成绩: " + scores[i]);
}
```

数组和 for 循环是绝配——用 `.length` 控制循环次数，用 `i` 当索引逐个访问元素。

### 六、找最大值

```java

int[] scores = {85, 92, 78, 63, 95};
int max = scores[0];                    // 先假设第一个最大
for (int i = 1; i < scores.length; i++) {
    if (scores[i] > max) {              // 如果找到更大的
        max = scores[i];                // 更新最大值
    }
}
System.out.println("最高分: " + max);   // 95
```

**七、找最小值**：把 `>` 换成 `<` 即可。
