### 一、变量是什么

变量 = 一块被命名的内存空间。声明变量时JVM做三件事：
1. 根据类型分配内存空间（如int分配4字节，byte分配1字节）
2. 把初始值的二进制形式存入这块空间
3. 把这块空间命名为变量名

### 二、八种基本数据类型

|数据类型|关键字|占用空间|取值范围|示例|
|---|---|---|---|---|
|字节型|`byte`|1 字节|-128 ~ 127|`byte b = 100;`|
|短整型|`short`|2 字节|-32768 ~ 32767|`short s = 30000;`|
|整型|`int`|4 字节|约 ±21 亿|`int i = 2000000000;`|
|长整型|`long`|8 字节|极大|`long l = 900000000000L;`|
|单精度浮点|`float`|4 字节|小数|`float f = 3.14f;`|
|双精度浮点|`double`|8 字节|更精确小数|`double d = 3.1415926;`|
|字符型|`char`|2 字节|单个字符|`char c = '中';`|
|布尔型|`boolean`|1 字节|true/false|`boolean b = false;`|

**记忆要点：**
-  整数默认是 `int` 类型，小数默认是 `double` 类型
    
- `long` 类型的值后面必须加 `L`，否则编译器会当成 `int`
    
- `float` 类型的值后面必须加 `f`，否则编译器会当成 `double`
